# Empathy Analyzer (RoBERTa)

Курсовий проєкт: Веб-застосунок для класифікації рівня емпатії у тексті.

## Технології
- Python, Flask
- PyTorch, Transformers (RoBERTa)
- SQLite, SQLAlchemy

## Як запустити
1. Встановіть залежності: `pip install -r requirements.txt`
2. Запустіть сервер: `python app.py`
